## Avant de commencer

Duration: 10

### Pré-requis

- Avoir une connection internet
- Git
- NodeJs

### Télécharger le repo

```bash
git clone  https://github.com/JKratus/Cypress-Devoxx-Hands-on-Lab.git
```

### Installer le projet

```bash
npm install
```

### Vérifier que le tout fonctionne

Lancer l'application

```bash
npm start
```

Lancer Cypress

```bash
npx cypress open
```
